### 安装依赖

<Step label="1">

使用 Homebrew 安装 Watchman：

<Terminal cmd={['$ brew install watchman']} />

</Step>

<Step label="2">

下载并安装 DevEco Studio。这是开发鸿蒙应用的官方 IDE。

访问[华为开发者联盟](https://developer.harmonyos.com/cn/develop/deveco-studio/)下载最新版本的 DevEco Studio。

</Step>

### 在鸿蒙真机上运行你的应用

<Step label="1">

**启用调试模式**

1. 在鸿蒙设备上打开"设置"
2. 进入"系统和更新"
3. 点击"开发人员选项"
4. 开启"USB调试"

</Step>

<Step label="2">

**通过 USB 连接你的设备**

通过 USB 将鸿蒙设备连接到计算机。

</Step>

<Step label="3">

在项目的根目录下运行以下命令：

<Terminal cmd={['$ npx xrn start']} />

根据提示，在命令行中输入 `h` 以在鸿蒙设备上运行你的应用。

</Step>
