### 在鸿蒙真机上运行你的应用

<Step label="1">

**启用调试模式**

1. 在鸿蒙设备上打开"设置"
2. 进入"系统和更新"
3. 点击"开发人员选项"
4. 开启"USB调试"

</Step>

<Step label="2">

**通过 USB 连接你的设备**

通过 USB 将鸿蒙设备连接到计算机。

</Step>

<Step label="3">

在项目的根目录下运行以下命令：

<Terminal cmd={['$ yarn start']} />

根据提示，在命令行中输入 `h` 以在鸿蒙设备上运行你的应用。

</Step>
