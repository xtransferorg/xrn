## 安装 Watchman 和 JDK

### 前提条件

使用像 [Homebrew](https://brew.sh/) 这样的包管理器安装以下依赖。

### 安装依赖

<Step label="1">

使用像 Homebrew 这样的工具 [安装 Watchman](https://facebook.github.io/watchman/docs/install#macos)：

<Terminal cmd={['$ brew install watchman']} />

</Step>

<Step label="2">

使用 Homebrew 安装名为 Azul Zulu 的 OpenJDK 发行版。此发行版提供适用于 Apple Silicon 和 Intel Mac 的 JDK。

在终端中运行以下命令：

<Terminal cmd={['$ brew install --cask zulu@17']} />

安装 JDK 后，在 **~/.bash_profile**（如果使用 Zsh，则为 **~/.zshrc**）中添加 `JAVA_HOME` 环境变量：

```bash
export JAVA_HOME=/Library/Java/JavaVirtualMachines/zulu-17.jdk/Contents/Home
```

</Step>
