<Step label="1">

### 安装 Xcode

打开 Mac App Store，搜索 [Xcode](https://apps.apple.com/us/app/xcode/id497799835)，点击 **安装**（如果已经安装，则点击 **更新**）。

</Step>

<Step label="2">

### 安装 Xcode 命令行工具

打开 Xcode，从 Xcode 菜单中选择 **设置...**（或按 <kbd>cmd ⌘</kbd> + <kbd>,</kbd>）。进入 **位置**，在 **命令行工具** 下拉菜单中选择最新版本来安装工具。

<ContentSpotlight
  alt="Xcode 偏好设置"
  src="/xrn/static/images/ios-simulator/xcode-command-line.png"
  className="max-w-[580px]"
/>

</Step>

<Step label="2">

### 安装 CocoaPods

打开终端，输入以下命令安装 CocoaPods：

<Terminal cmd={['$ brew install cocoapods']} />

</Step>
