### 安装鸿蒙开发工具


下载并安装 DevEco Studio。这是开发鸿蒙应用的官方 IDE。

请前往[下载中心](https://developer.huawei.com/consumer/cn/download/deveco-studio)，登录华为账号后下载DevEco Studio，并根据下载中心页面工具完整性指导进行完整性校验。
### 鸿蒙环境变量配置

在 .bash\_profile 或 .zshrc 下面添加配置

<SnackInline label='.bash_profile 或 .zshrc 配置'>
```shell
export PATH="/Applications/DevEco-Studio.app/Contents/sdk/default/openharmony/toolchains:$PATH" # 按照实际 SDK 安装路径配置，需要选择{显示包内容}
HDC_SERVER_PORT=7035 # HDC 端口变量名为：HDC_SERVER_PORT，变量值可设置为任意未被占用的端口，如 7035
launchctl setenv HDC_SERVER_PORT $HDC_SERVER_PORT 
export HDC_SERVER_PORT

# 鸿蒙配置
export RNOH_C_API_ARCH=1

# HamonyOS SDK
export HAMONY_TOOL_HOME="/Applications/DevEco-Studio.app/Contents"
export DEVECO_SDK_HOME="$HAMONY_TOOL_HOME/sdk"
export PATH="$HAMONY_TOOL_HOME/tools/ohpm/bin:$PATH"
export PATH="$HAMONY_TOOL_HOME/tools/hvigor/bin:$PATH"
export PATH="$HAMONY_TOOL_HOME/tools/node/bin:$PATH"
```
</SnackInline>
