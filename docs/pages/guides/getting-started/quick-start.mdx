你需要先 <A href="/guides/getting-started/basic-configuration">配置你的环境</A>，配置完成后，继续以下步骤创建应用并开始开发。

<Step label="1">
  创建应用
  <Terminal cmd={['$ npx @xrnjs/create']} />
</Step>

<Step label="2">
  构建一个 Android 应用
  <Terminal cmd={['$ yarn build:android']} />
</Step>

<Step label="3">
**运行你的应用**

进入 main-bundle 目录，并启动主 bundle：

<Terminal cmd={['$ cd main-bundle']} />
<Terminal cmd={['$ yarn start']} />

根据提示，<strong>在命令行中输入 `a`</strong> 以在 Android 设备上运行你的应用。

</Step>

<Step label="4">
**扫码调试项目**

根据提示，在命令行中输入 `p` 打印二维码，使用调试面板的扫码功能调试项目。
