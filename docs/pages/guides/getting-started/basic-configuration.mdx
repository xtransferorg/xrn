## 环境要求

需要使用 [Homebrew](https://brew.sh/) 安装一些依赖。

为了确保兼容性，需要安装 Node.js 18 版本。
[nvm 官方仓库](https://github.com/nvm-sh/nvm)

<Terminal cmd={['$ nvm install 18', '$ nvm use 18']} />


需要安装 [Yarn](https://yarnpkg.com/) 包管理器。

<Terminal cmd={['$ brew install yarn']} />

需要安装 [watchman](https://facebook.github.io/watchman/docs/install#macos)。

<Terminal cmd={['$ brew install watchman']} />
