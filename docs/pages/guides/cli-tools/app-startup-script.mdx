`xrn start`是一个应用启动工具，用于启动ReactNative服务并自动下载、安装、启动应用，提供各设备模拟器和真机支持，以及通过扫码快速关联服务ip的能力。

## 前置环境

请确保配置好安卓或iOS本地环境，参考：

<BoxLink
  title="基础配置"
  description="学习如何设置开发环境以开始 App 开发"
  href="/guides/getting-started/basic-configuration"
/>

## 安装

<Terminal
  cmd={['$ yarn add -D @xrnjs/cli']}
/>

## 使用

### 启动

在项目根目录下，运行以下命令：

<Terminal cmd={['$ npx xrn start']} />
